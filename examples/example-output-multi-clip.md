# 原创连续多条输出示例

> 为简洁展示，仅列出两条。实际任务按剧情继续扩展。

【项目模式】
连续多条4–15秒；真人影视级。

【全局剧情分析】
不可更改事实：雨夜旧书店；周遥从门外进入；发现林澈找到的旧日记；两句原台词按顺序发生。拆成两条：第01条建立空间并完成“进入与发现”，第02条完成“两句对白与日记推进”。

【分镜形式决策】
智能宫格，共6个必要镜头：第01条2格，第02条4格；对白近景和最终尾帧使用较大画格。镜头数不适合强制九宫格，镜头编号S01–S06跨条连续。

【镜头使用清单】
- S01–S02｜USE｜第01条0–8秒｜进入、发现与第01条尾帧；
- R01｜REFERENCE-ONLY｜第01条尾帧状态，服务S03起始继承，不单独生成、不占时长；
- S03–S06｜USE｜第02条0–12秒｜两句对白、日记推进与第02条尾帧；
- P01｜SKIP｜重复建立书店空间的备选镜头，不提交生成端。

【全局角色多视角设计】
- CHAR-001林澈：单独角色板图片1；正面、左右侧面、背面；窄长脸、低平眉、深眼窝、直鼻梁、清晰下颌线，短黑发，深灰衬衣。
- CHAR-002周遥：单独角色板图片2；正面、左右侧面、背面；柔和椭圆脸、略上扬眉眼、较短鼻梁、饱满唇形，深色长发，深色长外套。
- 两人的外轮廓、眉眼、鼻形、唇形和年龄感均形成稳定差异，不依赖服装区分。

【全局角色身份注册表】
- CHAR-001｜林澈｜图片1｜适用S01、S03–S06；
- CHAR-002｜周遥｜图片2｜适用S01–S06；
- 分镜格在人像附近标角色ID，信息栏重复角色名和图片编号；API content中图片1、图片2顺序不变。

【全局详细分镜图】
阅读顺序S01 → S02 → S03 → S04 → S05 → S06。执行图只放USE与服务S03的R01，不放P01。每格标注所属条目、时间码、景别、机位/焦段感、角色ID、站位、动作/运镜箭头、原台词/声音、道具、光向、转场和连续性锚点；S02标为第01条尾帧，R01记录该状态，S03完整继承后再切近景，S06标为第02条尾帧。

【全局角色锁定】
- 林澈：深灰衬衣、袖口微湿，始终位于柜台内侧
- 周遥：深色长外套，右手持收拢黑伞，从门口进入

【全局场景锁定】
- 空间结构：夜间旧书店，柜台居中，玻璃门右后方
- 轴线：两人关系轴连接柜台内侧与右后方门口，摄影机保持同侧
- 光影：冷蓝雨光从右后方进入，暖黄吊灯位于柜台上方

【全局声音规则】
有雨声、门铃、水滴与道具摩擦声；无音乐、无BGM、无字幕、无水印；台词口型同步。

【全局摄影与物理规则】
- 摄影基线：宽银幕，32–65mm等效焦段，固定/缓慢轨道移动，拉焦只由日记或说话人触发
- 运动基线：真实重力、脚底接触、湿物摩擦与水滴运动
- 表演基线：克制，先视线与呼吸变化，再说话或推动道具

【第01条｜8秒｜周遥进入并发现日记】

【转场类型】
项目起始；建立初始状态。

【起始帧继承】
项目起始帧。摄影机在柜台外侧偏右，中景平视；林澈在左中景低头压住日记；玻璃门关闭，周遥位于门外。

【导演方案（审阅用，不粘贴到生成器）】
状态差分：门由关闭变为打开再关闭；周遥由门外移动到门内；其余状态连续。

【可复制图片提示词】
雨夜旧书店中景，林澈在柜台内侧压住一本湿日记，周遥刚从右后方玻璃门进入，右手持黑伞，冷蓝雨光与暖黄吊灯形成真实混合光。

【可复制视频提示词】
即梦Seedance视频，本条完整8秒，有音效，无音乐，无BGM，无字幕，无水印。除时间轴明确变化外，其余状态保持连续。0–3秒玻璃门向内打开，门铃轻响，周遥持黑伞进入一步；3–6.5秒摄影机缓慢前推，她停在门内，视线落到柜台上的湿日记；6.5–8秒她肩膀轻微僵住，雨声与伞尖滴水声清晰，没有台词。真人模式使用自然肤质、50mm等效焦段观感、真实水滴与脚底接触。

【尾帧接力合约】
摄影机位于柜台外侧偏右，中景；周遥站在右后方门内，右手垂直持伞，视线看向左前方日记；林澈在左中景低头，右手压住日记；门已关闭；冷蓝雨光与暖黄吊灯保持不变。

【状态账本更新】
周遥位置由门外变为门内；玻璃门回到关闭；其他字段不变。

【负面约束】
禁止黑伞换手、脚底滑行、门体穿模、视线错位和水滴逆重力。

【第02条｜12秒｜两句对白与日记推进】

【转场类型】
同场切镜；继承角色、道具、空间与光影，允许摄影机切至近景。

【起始帧继承】
完整复现第01条尾帧：摄影机、人物位置、黑伞右手状态、日记位置、门的关闭状态和冷暖光方向均不改变。

【导演方案（审阅用，不粘贴到生成器）】
状态差分：摄影机由中景切至两次近景；日记由林澈右手下方移动到柜台前缘。

【可复制图片提示词】
雨夜旧书店内，林澈侧前方近景，右手把湿日记推向柜台前缘；周遥在右后景凝视日记，黑伞仍在右手，冷蓝雨光与暖黄吊灯方向连续。

【可复制视频提示词】
即梦Seedance视频，本条完整12秒，有音效，无音乐，无BGM，无字幕，无水印。除时间轴与同场切镜合同明确变化外，其余状态连续。0–4秒切周遥近景但保持同侧轴线，她低声说：“你还是把它找回来了。”口型同步；4–5秒静默，只听雨声；5–10.5秒切林澈侧前方近景，他不抬头，右手将日记缓慢推到柜台边缘，同时说：“我找回来的，不只是它。”；10.5–12秒台词已结束，镜头停止，日记停在柜台边缘，周遥在后景盯着日记。真人模式保持自然肤质、真实湿物摩擦、明确焦点路径和克制表演。

【尾帧接力合约】
摄影机停在林澈侧前方近景；林澈低头，右手停在日记后方；日记位于柜台前缘；周遥仍在右后方，黑伞在右手；门关闭；光影与雨声持续。下一条开头必须先复现此状态。

【状态账本更新】
日记从林澈右手下方移动到柜台前缘；摄影机变为林澈侧前方近景；其他字段不变。

【负面约束】
禁止改动原台词、日记无摩擦滑动、黑伞换手、镜头越轴和口型错配。
