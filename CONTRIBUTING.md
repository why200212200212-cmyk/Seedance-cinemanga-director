# Contributing

感谢参与 `Seedance Cinemanga Director`。

## 提交方向

欢迎贡献：

- 更稳定的镜头与连戏规则；
- 不同题材的原创示例；
- 更准确的对白时长模型；
- 3D国漫与真人影视模式优化；
- Seedance 实测中可复现的问题与修复；
- 文档、校验脚本和安装说明改进。

## 基本原则

贡献内容不得：

- 擅自改写用户原台词；
- 默认新增无关人物、动物、道具或剧情；
- 用空泛画质词替代可执行导演设计；
- 引入未经许可的剧本、小说、图片或角色设定；
- 包含密钥、账号、私人素材或未脱敏数据。

## 本地检查

```bash
python scripts/validate_skill.py
```

## Pull Request 建议

PR 描述请写清：

- 改了什么；
- 为什么需要；
- 对单条15秒和连续多条模式的影响；
- 使用什么输入样例验证；
- 是否改变现有输出格式。
