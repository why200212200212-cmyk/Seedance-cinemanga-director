## 一、固定镜头（Static Shot）

**定义**：摄影机锁死在三脚架上，画面内只有被摄体运动。

**功能**：
- 对话场景中让观众专注于表演和台词
- 极度克制的紧张感（罗宏涛式静默压迫）
- 为后续的微小运镜提供"静"的基准，动才更有力

**电影案例**：《窃听风暴》审讯室对话、《十二怒汉》全片

**AI提示词**：
```
static shot, locked down tripod, no camera movement, [subject] moves within frame
```

**在文档中的使用**：地牢对话场景中利用固定镜头配合三角形外反拍，形成无法逃脱的牢笼感。

---

## 二、横摇（Pan）

**定义**：摄影机左右水平旋转（机身位置不动，机头转动）

**分类**：
- **跟随摇（Follow Pan）**：追随运动的被摄体
- **揭示摇（Reveal Pan）**：从左到右/从右到左揭示新信息
- **连接摇（Connection Pan）**：在两个对话者之间切换建立空间关系

**功能**：
- 揭示新信息（横摇展现大场景全貌）
- 跟随运动主体建立动势
- 加快/减慢摇速控制节奏

**AI提示词**：
```
pan left to right, slow pan revealing [scene], horizontal camera rotation
pan right to left following [subject]'s movement
```

---

## 三、甩镜（Whip Pan / Swish Pan）

**定义**：极快速地横摇，画面产生强烈运动模糊

**功能**：
- 快速转场，连接两个空间或时间
- 增加活力感、节奏感
- 在角色之间建立能量连接

**电影案例**：《爱乐之城》中男女主角初遇时的甩镜连接

**AI提示词**：
```
whip pan, fast camera rotation, motion blur transition, energetic camera movement
```

---

## 四、纵摇（Tilt）

**定义**：摄影机上下垂直旋转

**分类**：
- **上摇（Tilt Up）**：从下往上，揭示崇高、权力、宏大
- **下摇（Tilt Down）**：从上往下，揭示脆弱、渺小、细节

**功能**：
- 展现垂直空间的高度（摩天大楼、巨人）
- 权力反转的视觉叙事（从低到高的人物关系揭示）
- 情绪升降的物理外化

**经典用法**：《侏罗纪公园》中博士们第一次看到恐龙时的Tilt Up

**AI提示词**：
```
tilt up, camera tilting upward revealing [subject], sense of scale
tilt down, camera tilting downward, looking down at [subject], vulnerability
```

---

## 五、推镜（Push-In / Dolly In）

**定义**：摄影机物理靠近被摄体

**功能**：
- 强化紧张感、近距离压迫
- 揭示角色的内心觉醒或决定
- 将观众拉入角色的心理世界

**推镜 vs 变焦推（Zoom In）**：

| 特性 | 推镜（Dolly In） | 变焦推（Zoom In） |
|---|---|---|
| 透视关系 | 改变（背景扩张） | 不变（背景放大） |
| 视觉感受 | 自然、沉浸 | 诡异、人工 |
| 用法 | 情感靠近 | 注意聚焦、突兀效果 |

**AI提示词**：
```
push in, dolly in, camera moving closer to [subject], intensifying tension
dolly zoom, vertigo effect, unsettling perspective change
```

**在文档中的使用**：逼婚段落的推镜用于强化母亲的权威施压。

---

## 六、拉镜（Pull-Out / Dolly Out）

**定义**：摄影机物理远离被摄体

**功能**：
- 制造孤立感、疏离感
- 揭示更广阔的环境信息
- 情绪的"冷却"或"放手"

**经典案例**：《闪灵》结尾拉镜揭示杰克冻死的命运

**AI提示词**：
```
pull out, dolly out, camera moving away from [subject], isolation, revealing environment
```

---

## 七、变焦（Zoom）

**定义**：不移动摄影机，通过改变镜头焦距改变视场

**分类**：
- 变焦推（Zoom In）：放大主体，压缩空间
- 变焦拉（Zoom Out）：缩小主体，展现场景

**为何特殊**：变焦没有人类视觉的对应物（人眼不会"zoom"），因此会产生不自然、诡异感

**在恐怖片中的应用**：库布里克在《闪灵》《全金属外壳》中使用zoom制造不安

**AI提示词**：
```
zoom in, slow zoom, focal length change, magnifying subject
zoom out, revealing shot, expanding frame
```

---

## 八、滑动变焦（Dolly Zoom / Vertigo Effect）

**定义**：摄影机推/拉同时反向变焦，保持主体大小不变而背景扭曲

**效果**：
- 极度不安、眩晕感
- 内心冲突的外化
- 超自然力量的压迫

**经典案例**：《迷魂记》楼梯井、《大白鲨》海滩、《指环王》甘道夫面对炎魔

**AI提示词**：
```
dolly zoom, vertigo effect, subject stays same size while background distorts, unsettling
```

---

## 九、横移（Truck / Tracking）

**定义**：摄影机平行于被摄体横向移动

**分类**：
- **顺移（Truck Right/Left）**：与角色平行移动
- **前移（Track In）**：跟随角色前进方向

**功能**：
- 跟随运动角色（最自然的跟随方式）
- 展现环境纵深
- 制造"同步感"（观众与被摄体一起移动）

**AI提示词**：
```
trucking shot, camera moving parallel to [subject], lateral tracking
tracking shot, camera following [subject] from side
```

---

## 十、升降（Boom / Crane）

**定义**：摄影机整体升高或降低

**分类**：
- **Boom Up**：整体升高（不同于Tilt，位置变高而非角度变高）
- **Boom Down**：整体降低
- **Crane Shot**：大型升降机，大范围垂直+水平综合运动

**功能**：
- 高度的权力隐喻（高=权力，低=屈从）
- 揭示大场面
- 从微观卷入宏观

**AI提示词**：
```
boom up, camera rising, gaining height, revealing more of scene
crane shot, sweeping camera movement, grand reveal
```

**在文档中的使用**：索菲亚站在画右大门高位 vs 奈雅跪缩在画左地表低位——这种垂直权力几何是Boom/Crane的静态等价物。

---

## 十一、弧线运镜（Arc Shot）

**定义**：摄影机围绕被摄体做弧线运动

**功能**：
- 360°审视角色
- 注入动感与能量
- 表示时间流逝或围绕核心主题

**AI提示词**：
```
arc shot, camera circling [subject], orbiting movement, dynamic
```

---

## 十二、手持摄影（Handheld）

**定义**：摄影机由摄影师手持/肩扛，刻意保持轻微摇晃

**功能**：
- 纪录片真实感、战争临场感
- 混乱、危险、不稳定情绪
- 主观代入（观众仿佛在现场）

**经典案例**：《拯救大兵瑞恩》奥马哈海滩、《谍影重重》打斗

**AI提示词**：
```
handheld, shaky camera, documentary style, raw, immersive, chaotic
```

---

## 十三、斯坦尼康/稳定器（Steadicam / Gimbal）

**定义**：使用身体稳定器实现流畅平滑的运动

**功能**：
- 长镜头无缝运镜（一镜到底）
- 紧跟角色穿行复杂空间
- 幽灵般的流畅移动（制造不安或梦境感）

**经典案例**：《好家伙》夜总会入场、《1917》伪一镜到底

**AI提示词**：
```
steadicam shot, smooth camera movement, flowing, gliding through space
one-shot sequence, continuous take, no cuts
```

---

## 十四、主观视角（Point-of-View / POV）

**定义**：摄影机模拟角色的眼睛，观众看到的就是角色看到的

**功能**：
- 极致代入感
- 限知视角制造悬念
- 揭示角色生理状态（醉酒POV、模糊POV、低血糖POV）

**AI提示词**：
```
POV shot, first person, subjective camera, seeing through [subject]'s eyes
```

---

## 十五、摇臂镜头（Jib / Crane）

**定义**：使用摇臂实现升降+横移的组合运动

**功能**：
- 从低到高或从高到低的平滑移动
- 建立空间的整体到局部/局部到整体

**AI提示词**：
```
jib shot, camera rising and moving forward, combining boom and dolly
crane shot, grand sweeping movement
```

---

## 十六、机械臂镜头（Technocrane / Robotic Arm）

**定义**：大型机械臂控制的摄影机，可以到达任何高度和角度

**功能**：
- 三维空间内的任意运动轨迹
- 穿越极狭窄空间的精密运动

## 运镜的情绪映射表（AI导演必需）

| 情绪目标 | 推荐运镜 | 原因 |
|---|---|---|
| 紧张/压迫 | Push-In, Handheld | 物理靠近 + 不稳定感 |
| 孤立/疏离 | Pull-Out, High Angle | 远离 + 俯视 |
| 混乱/恐慌 | Whip Pan, Handheld, Dutch Angle | 快速摆动 + 摇晃 + 倾斜 |
| 权力/崇高 | Low Angle Tilt Up, Boom Up | 仰视 + 升高 |
| 屈辱/无助 | High Angle, Boom Down | 俯视 + 降低 |
| 亲密/沉浸 | Push-In, POV, Close-Up | 靠近 + 主观 + 局部 |
| 悬念/不安 | Slow Pan, Dolly Zoom, Static | 缓慢揭示 + 透视扭曲 + 静止 |
| 节奏/活力 | Whip Pan, Arc, Handheld | 快速 + 环绕 + 摇晃 |
| 史诗/宏大 | Crane, Wide Crane Pan | 升降 + 横摇 + 全景 |
| 梦境/迷幻 | Steadicam Slow Push, Dolly Zoom | 流畅 + 透视扭曲 |

## AI导演的运镜选择原则

对于AI导演智能体，运镜选择应遵循以下决策树：

```
1. 本镜的叙事功能是什么？
   → 传达情绪？揭示信息？建立空间？跟随动作？

2. 谁在主导这个镜头？
   → 角色视角（POV/主观）？观察者视角（客观）？全知视角（上帝）？

3. 空间约束是什么？
   → 密闭空间（固定/推/拉）？开放空间（摇/移/升降）？

4. 与前/后镜的关系？
   → 是否需要匹配剪辑（Action Match）？是否遵循轴线？

5. AI生成能力约束是什么？
   → 是否属于物理接触死穴？是否需要死穴规避方案？
```
