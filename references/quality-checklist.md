# 最终质量检查清单

输出前逐项检查。任一项不通过，先修正再提交。

## A. 剧情与人物

- [ ] 剧情因果和事件顺序与原文一致；
- [ ] 原台词逐字一致；
- [ ] 没有新增无关人物、动物、场景、道具和宏观动作；
- [ ] 当前段落必须出现的人物全部存在；
- [ ] 无功能角色已排除；
- [ ] 台词说话人和声音来源正确。

## B. 时间与节奏

- [ ] 单条模式总时长严格15秒；
- [ ] 多条模式每条4–15秒；
- [ ] 台词朗读时长合理；
- [ ] 停顿、呼吸、反应和动作有时间；
- [ ] 最后0.5–1.5秒用于收束；
- [ ] 结尾没有卡住半句台词。

## C. 镜头与表演

- [ ] 景别和机位服务剧情；
- [ ] 运镜路径物理可行；
- [ ] 人物站位、面对关系和屏幕方向明确；
- [ ] 动作顺序具体可执行；
- [ ] 手部状态和道具使用关系正确；
- [ ] 无穿墙、穿模、瞬移和无依据慢动作；
- [ ] 焦点变化与主体动作匹配。

## D. 连续性

- [ ] 角色脸型、发型、服装、体型一致；
- [ ] 场景结构、门窗和家具位置一致；
- [ ] 道具数量、位置、朝向和状态一致；
- [ ] 左右手没有无故交换；
- [ ] 光源方向、色温、阴影和反光一致；
- [ ] 人物视线和屏幕方向一致；
- [ ] 未发生无解释越轴；
- [ ] 多条模式的起始帧完整继承上一条尾帧。

## E. 声音与纯净度

- [ ] 环境音和动作音与画面同步；
- [ ] 默认无音乐、无BGM；
- [ ] 默认无字幕、无画面文字；
- [ ] 默认无水印、无Logo；
- [ ] 内心独白人物不张嘴；
- [ ] 电话、门外、广播和远处声源有空间属性。

## F. 输出格式

- [ ] 使用正确模板；
- [ ] 单条模式包含 `_::~OUTPUT_START::~_` 和 `_::~OUTPUT_END::~_`；
- [ ] 多条模式每条包含起始继承和尾帧接力合约；
- [ ] 图片提示词是静态代表画面；
- [ ] 视频提示词是连续动态描述；
- [ ] 没有输出隐藏推理或无关解释。

## G. 影视真实感硬门槛

- [ ] 每个动作都有触发、路径、受力和落点，没有无因提前反应；
- [ ] 每个运镜都有唯一主要目的、可实现路径和明确停止构图；
- [ ] 焦段、主体距离、景深、拉焦与画面透视不冲突；
- [ ] 主光、高光、阴影、反射、雨雾和体积光都有可解释来源；
- [ ] 人物重心、脚底接触、手与道具接触、衣发次级运动符合物理；
- [ ] 表情由视线、呼吸、面部和姿态微动作组成，不是僵硬摆拍；
- [ ] 没有用画质词堆叠替代具体摄影、表演和美术指令；
- [ ] 单镜复杂度在可控范围，必要时已经拆镜或拆条。

## H. 模式专项

### 3D国漫影视级

- [ ] 角色身份锚点、服装层、材质参数和纹样位置稳定；
- [ ] 皮肤、头发、布料、金属和玻璃的材质响应互不混淆；
- [ ] 动作含预备、发力、位移、缓冲和余韵，次级运动合理衰减；
- [ ] 无脸型漂移、材质闪烁、几何呼吸、肢体穿模和道具悬浮。

### 真人影视级

- [ ] 摄影设备、焦段、机位、路径和现场空间相容；
- [ ] 曝光、运动模糊、景深和焦点变化符合镜头条件；
- [ ] 肤质、妆发、服装、汗水、雨水、污损和伤势状态连续；
- [ ] 无蜡像脸、脚底滑行、果冻形变、焦点抽搐和曝光泵动。
