# 武术与影视动作参考来源

本文件只提供按需调用的来源摘要，不覆盖原剧本、角色设定、主工作流、平台能力或现场专业人员判断。访问日期：2026-07-16。

## 1. 武术套路与项目结构

- [IWUF：Taolu 项目说明](https://iwuf.org/en/sport-wushu/competitive-wushu/taolu/)：把套路定义为按原则编排、连续连接的预定技法，并区分个人、对练和集体项目；评价维度包括动作质量、整体表现和难度。用于建立“动作质量—整体节奏—难度可执行性”的三层检查，不照抄竞赛套路。
- [IWUF：2024武术套路与散打竞赛规则发布](https://www.iwuf.org/en/news/gjwl/2024/0914/8955.html)：新规则由技术委员会编制并于2025年实施。需要竞赛术语或套路分类时优先查当前规则PDF，不使用旧版固定分值作为影视规则。
- [IWUF：2019套路规则摘录](https://www.iwuf.org/wp-content/uploads/2020/03/IWUF-Wushu-Taolu-Competition-Rules-Judging-Methods-Excerpt-2019.pdf)：包含动作质量、整体表现、难度，以及对练和集体项目的评分结构。仅用于理解评价维度；平台和竞赛规则更新时以最新版为准。

## 2. 影视与舞台动作安全

- [Contract Services：Safety Bulletin #4 Stunts](https://www.csatf.org/wp-content/uploads/2018/05/04STUNTS.pdf)：强调由合格特技协调/专业人员负责计划、场地、测试、排练、修改、人员与设备；演员应有咨询机会并同意参与。用于把真实拍摄输出限定为制作备注和专业复核。
- [Contract Services：Safety Bulletins 索引](https://www.csatf.org/production-affairs-safety/safety-bulletins/)：提供特技、枪械、车辆、火焰、坠落等专题入口。高风险元素只路由到专业部门，不由本Skill给执行步骤。
- [Actsafe：Stage Combat, Stunts & Weaponry](https://actsafe.ca/topic/stunts/09-stage-combat-stunts-weaponry/)：面向表演艺术行业的舞台格斗、特技和道具安全建议。用于支持“排练、受控道具、风险沟通、专业监督”的边界。

## 3. 动作物理与运动研究

- [PLoS One / PubMed：不同项目横踢生物力学比较](https://pubmed.ncbi.nlm.nih.gov/28841670/)：使用3D运动捕捉和力数据比较不同武术项目的横踢运动学，说明动作节奏、膝髋速度和重心变化具有项目差异。用于避免所有角色共享同一动作模板。
- [Sports / PubMed：前踢与横踢动力学系统综述](https://pubmed.ncbi.nlm.nih.gov/37624121/)：汇总速度、力量与水平差异。用于提示动作应写清速度目的、身体链和恢复，而不是堆“迅猛有力”。
- [Frontiers / PubMed：武术动作的角动量模型](https://pubmed.ncbi.nlm.nih.gov/34375352/)：讨论旋转、摩擦、身体轴和角动量。用于画面物理中的支撑脚、转轴、地面反作用和旋转余韵，不用于计算现实伤害。
- [Scientific Data / PubMed：空击、靶击与对手条件的动作捕捉数据](https://pubmed.ncbi.nlm.nih.gov/33462240/)：包含不同水平运动者的拳腿动作捕捉。用于理解同一动作在无目标、静态目标和对手条件下会改变身体段落与时机。

## 4. 使用规则

- 优先提取结构、评估维度、空间与物理原则，不复制竞技计分、训练处方或现实伤害参数；
- 武术流派描述必须服务人物背景和叙事，不根据单篇论文给角色贴绝对标签；
- 真人制作安全由当地法律、制片安全规范和合格特技人员决定；本文不是现场安全认证；
- 生成端无法稳定执行多人高速接触时，降低并发、拆节拍、增加空间锚点或拆VID，不用危险细节换取“真实感”。
