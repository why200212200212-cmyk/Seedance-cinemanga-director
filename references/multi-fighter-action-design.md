# 多人格斗与影视动作设计

仅在原剧本包含两人以上肢体冲突、追逐中的近身动作、武术套路、兵器表演或用户明确要求动作设计时读取。先完成原有剧情分析、必要人物过滤、角色注册和状态账本，再使用本模块；不得为了展示动作而新增打斗。

## 1. 模块边界

本模块设计的是影视画面、表演节拍、空间调度和生成提示词，不是现实格斗教学。不得提供伤害弱点、致残方法、实战攻击教程、武器制作或绕过现场安全的建议。真人拍摄必须由合格动作指导/特技协调员根据演员能力、场地、道具和当地规定重新评估。

## 2. 动作角色图

为每名参与者保留永久 `CHAR-ID`，再登记动作身份：

`FTR-ID → CHAR-ID → 阵营/关系 → 叙事目标 → 动作风格 → 能力上限 → 当前伤势/疲劳 → 道具 → 左右手 → 禁止动作`

每个节拍明确：

- **主动者**：谁触发动作；
- **受动者**：谁观察、回避、格挡、失衡或反应；
- **空间功能者**：谁封路、护人、争夺道具、撤离或改变地形；
- **非参与者**：本节拍不进入动作线，不得无故冲入画面。

多人场面使用 `TEAM-ID` 和 `FTR-ID`，禁止用“众人”“其他人”代替身份。每一次接近、接触、摔倒和离场都绑定唯一双方及镜头时间。

## 3. 空间拓扑与并发控制

先画俯视动作地图，再写动作：

- 标出入口、出口、掩体、台阶、家具、危险区、摄影机安全区和不可穿越边界；
- 为每人记录 `ZONE-A/B/C`、屏幕左/右、前中后景、面对方向和移动路线；
- 使用独立路线 `ACT-A/B/C`，交叉点写明先后顺序和时间，不允许多人同刻穿过同一空间；
- 每个短节拍只保留一个主视觉动作和最多一个次级动作；复杂并发拆成连续节拍或拆镜；
- 非当前主动者必须有可见目的：重新站位、寻找出口、保护目标、恢复平衡或被环境阻隔，不能排队等待攻击。

建议用“行动令牌”控制可读性：普通4–15秒镜头同一时刻通常只让1名主动者发起主要动作；确有剧情需要时允许2条动作线，但必须分前后景、方向和时间相位。

## 4. 影视动作节拍

每个动作单元按画面可读性写成：

`意图/预备 → 发起 → 路径 → 回避/格挡/受力反应 → 重心与落点 → 余韵 → 新状态`

- **意图/预备**：视线、呼吸、肩胯、脚步或道具起始状态；
- **路径**：方向、层级、高低变化、运动弧线和空间目标；
- **反应**：先看到/感知，再回避或承受，不允许提前反应；
- **重心与落点**：支撑脚、身体轴、接触面、停止位置；
- **余韵**：衣发、尘土、道具、呼吸和恢复动作；
- **新状态**：位置、朝向、伤势、疲劳、道具归属和下一节拍入口。

屏幕冲击优先使用错位、遮挡、反应时差、声音和剪辑制造，不要求真实击打。将接触标为 `NO-CONTACT`、`CHEATED-CONTACT` 或由专业人员现场决定的 `CONTROLLED-CONTACT`；AI提示词只描述银幕效果和视觉间距，不指挥真人实际接触。

## 5. 武术套路影视化

先选择一种主风格，不把拳种名称当装饰词堆叠。套路结构用于动作语言和节奏，不改变人物背景：

- **长拳类视觉倾向**：展开、长线、速度、腾挪和较大空间路径；
- **南拳类视觉倾向**：稳定根基、短促节奏、清晰停顿和强烈整体发力观感；
- **太极类视觉倾向**：连续圆弧、重心转移、开合、黏随感和快慢对比；
- **对练/Duilian**：预先编排的攻防关系，重点是双方时机、距离和互相成就；
- **集体/Jiti**：队形、同步、错拍、镜像与空间图案，不等同于所有人同时围攻一人；
- **器械表演**：先锁定道具ID、持有者、长度感、运行平面和安全净空，再设计路线。

单条4–15秒视频只选1–2个可识别动作短句：

`起势轮廓 → 位移与技法簇 → 方向/层级变化 → 人物关系节拍 → 高潮动作 → 收势/接力状态`

除非原剧本明确混合传承或人物经历，同一角色保持一个主动作体系。跨体系混合时写明剧情依据、主次比例和视觉边界，不做随机“百家拳法”拼接。

### 5.1 叙事化套路短句库

以下只定义银幕节奏和人物关系，不规定现实攻击目标或实战方法。根据剧情选一条并改写：

- **双人试探**：建立距离 → 一方显露意图 → 另一方错位化解 → 双方交换位置 → 短暂停顿重估；
- **强弱逆转**：强者控制空间 → 弱者被迫移动 → 环境/信息改变 → 弱者夺回一个空间锚点 → 新对峙；
- **一对多可读轮转**：群体封锁路线 → 一条主动作线进入 → 主角脱离当前关系 → 第二条动作线在不同层级接续 → 全员重新落位；
- **护送/守点**：保护目标建立 → 威胁从一侧逼近 → 保护者改变站位与视线 → 目标移动至新ZONE → 防线重组；
- **追逐中的短冲突**：共同移动方向 → 障碍改变节奏 → 一次短接触幻觉 → 双方分离 → 追逐方向继续；
- **双人对练展示**：同步起势 → 镜像/错拍技法 → 角色风格形成对比 → 同一高潮时点 → 清晰收势；
- **集体套路**：队形建立 → 同步段 → 分组错拍 → 交叉但分时通过 → 队形重组与统一收束；
- **器械视觉段**：道具轮廓建立 → 单人净空路径 → 另一人进入不同运行平面 → 遮挡/声音形成银幕交锋 → 道具归属与安全距离回到明确状态。

如果短句在4–15秒内需要超过两个主要关系变化、两条同时交叉路线或多个高难度动作，把它拆成多个VID，不压缩成不可读混战。

## 6. 镜头与剪辑

- 建立镜头先交代人数、阵营、出口和空间轴线；
- 每个动作高潮只设一个主阅读点，近景用于表演/接触错位，广角用于路线/队形；
- 切点优先放在动作发起、遮挡峰值、方向改变、落点或反应完成处；
- 保持屏幕方向和人物身份，不用高速摇镜、碎切或特效掩盖无法理解的动作；
- 主观镜头只在叙事需要时使用，并明确视线来源；
- 多人混战每2–4个节拍重新给出空间锚点，但不要机械插入大全景；
- 伪一镜到底必须记录隐藏接缝、遮挡物、动作相位、速度、焦点和曝光锚点。

## 7. 双模式专项

### 3D国漫影视级

- 为每名角色分别锁定骨骼、关节极限、衣发层、武器碰撞体和次级运动；
- 多人接触前先分离动作层和碰撞顺序，避免肢体互穿、复制角色和道具瞬移；
- 夸张动作仍保留预备、发力、空中轨迹、落地缓冲和恢复，不把速度线当成物理；
- 群体动作使用不同节奏、轮廓和位置，避免镜像复制同一套动画。

### 真人影视级

- 根据演员/替身、服装、地面、道具、机位和可用排练时间降低动作复杂度；
- 使用镜头角度、景别、遮挡、表演反应和声音完成“银幕命中”，不要求真实命中；
- 记录动作指导、慢速走位、停止信号、保护装备和安全区域为制作备注，但不把这些文字放进成片提示词；
- 枪械、爆破、火焰、车辆、高坠、钢丝、动物和真实刃具只标为专业特技/专业部门处理，本Skill不设计执行方案。

## 8. 输出合同

动作场景在原单条/多条模板中额外输出：

1. `动作叙事目标`：冲突如何改变人物或剧情；
2. `动作角色图`：FTR-ID、阵营、目标、风格、能力与状态；
3. `俯视空间图`：ZONE、路线、交叉时序、危险区和机位；
4. `动作节拍表`：时间、主动/受动者、动作链、镜头、接触幻觉、声音、结束状态；
5. `套路短句`：主风格、起势、技法簇、关系节拍、高潮、收势；
6. `安全与可执行性备注`：仅供制作审阅；
7. `Seedance可复制提示词`：按时间线保留身份、空间、动作、摄影和连续性，不包含现场危险操作指令。
