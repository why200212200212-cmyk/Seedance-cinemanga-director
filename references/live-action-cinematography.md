# 真人影视级摄影与表演规范

## 1. 摄影机合同

每个镜头明确：画幅、景别、等效焦段、机位高度、俯仰/偏航、主体距离、运动设备、路径、速度变化、焦点对象和停止构图。

- 18–28mm：空间、速度与压迫；靠近人物会产生明显透视，慎用于面部特写。
- 32–50mm：自然叙事与走位，适合中景、双人和跟拍。
- 65–100mm：情绪隔离与压缩空间，要求更稳定的对焦和机位。
- 手持只在主观不稳或现场压力中使用；稳定器、轨道、摇臂和肩扛需符合现场空间。

不要同时要求互斥的设备行为，例如“强烈手持抖动且绝对稳定”“长焦压缩且夸张广角透视”。

## 2. 曝光、景深与运动

- 先确定帧率与运动意图，再决定快门观感；常规动作保留自然运动模糊，高速信息镜头可适度缩短曝光但避免频闪般生硬。
- 景深由焦段、光圈、距离和传感器观感共同决定；对话反打保持可接受的景深和视线落点。
- 拉焦必须有叙事触发对象与明确起止焦面；不得无因抽动或同时锁定不同距离主体。
- 曝光变化只能来自进入不同光区、灯具变化或摄影机自动/人工调整，并说明过渡。

## 3. 灯光与色彩

- 每个可见高光都对应合理光源；固定主光方向、色温、软硬、衰减和环境反弹色。
- 夜景保留暗部层次，不把黑位抬成均匀灰，也不让人物无来源发亮。
- 雨夜反射遵循路面湿润度、光源位置和观察角；烟雾与体积光只在逆光或侧逆光中明显。
- 肤色在不同色温下保持生命感，避免橙青滤镜覆盖所有局部颜色。

## 4. 演员调度

- 把情绪词翻译为可表演行为：视线停留、眨眼频率、呼吸、吞咽、下颌紧张、肩颈张力、重心移动、说话前后的反应。
- 动作使用“意图—决定—执行—后果”链；演员不能在没有看到、听到或触碰触发物前提前反应。
- 对话中未说话者仍保持倾听行为，但不抢戏；多人镜头明确谁处于叙事焦点。
- 保持真实接触：脚落地、手握物、衣物受风、身体碰撞和重量转移必须连贯。

## 5. 美术、妆发与现场状态

- 锁定服装层次、扣合状态、袖口、污渍、雨水、汗液、血迹、伤口、妆容和发丝湿度。
- 场景磨损与生活痕迹应有因果，不用随机垃圾、烟雾或火花制造“电影感”。
- 可读文字、品牌、车牌、屏幕UI和标识仅在剧情必要时出现；否则避免生成不可控乱码。

## 6. 失败抑制

明确禁止：蜡像皮肤、五官漂移、眼神不对位、口型与声源错配、手指融合、身体穿透、脚底滑行、无重力动作、焦点抽搐、曝光泵动、果冻形变、背景人复制、反射中人物缺失、雨水方向突变和无来源慢动作。
