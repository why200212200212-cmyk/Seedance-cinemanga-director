# 双影视风格模式

## 精致3D国漫影视级

### 必须具备

- 影视级3D国漫CG，不是真人照片；
- 比例可信，非低幼、非Q版、非塑料玩具感；
- 面部细节、毛发分组、衣料褶皱和材质稳定；
- 表情与动作具备动漫级可读性，但不过度夸张；
- 场景和物理关系真实可信；
- 光影、色彩和镜头语言具有电影层次。

### 避免

- 仿真人皮肤毛孔与原始照片感；
- 低幼卡通、超大眼、过短肢体；
- 角色脸部频繁变化；
- 大量引擎名、分辨率词和空泛画质词堆叠。

## 真人影视级

### 必须具备

- 真实演员质感和自然皮肤纹理；
- 服装、妆发、环境磨损和天气细节可信；
- 真实摄影机能够完成的运镜；
- 演员动作符合重力、惯性和空间；
- 自然表演，不像广告定格或短视频摆拍；
- 真实环境音和动作音。

### 避免

- 蜡像脸、过度磨皮和不自然高光；
- 游戏CG、虚假景深和无理由慢动作；
- 镜头无物理限制地穿越物体；
- 服装、污渍、伤痕和天气状态跳变。

## 风格优先级

1. 用户明确风格；
2. 用户提供的参考图；
3. 项目已建立的角色与世界观；
4. 本 Skill 默认风格。
