## 一、构图的核心目的

构图不是为了"好看"，而是为了**引导观众视线 + 传达潜台词 + 创造情绪**。

---

## 二、七大经典构图法则

### 2.1 三分法（Rule of Thirds）

**定义**：将画面用2条横线和2条竖线分成9宫格，关键元素放在交叉点上。

**功能**：
- 将角色放在画框一侧——暗示被世界隔离
- 将角色放在画框另一侧——暗示与环境的关系发生变化

**经典案例**：《夜行者》中Lou Bloom永远在画框边缘、只露半张脸——难以信任、遥远

**AI实现**：
```
rule of thirds, subject on left third, negative space on right
```

### 2.2 对称构图（Symmetrical Composition）

**定义**：画面左右完全对称，主体居中。

**功能**：
- 庄重、权力、仪式感（库布里克式）
- 打破对称时产生强烈的不安

**经典案例**：《2001太空漫游》《闪灵》走廊、《布达佩斯大饭店》

**AI实现**：
```
symmetrical composition, centered, perfectly balanced, Kubrick symmetry
```

### 2.3 引导线（Leading Lines）

**定义**：画面中的实线或虚线引导观众视线指向主体。

**类型**：
- 对角线（动态、不安）
- 水平线（平静、压抑）
- 垂直线（崇高、权力）
- S型曲线（优雅、神秘）
- 汇聚线（纵深感、无限）

**AI实现**：
```
leading lines, diagonal lines drawing eye to subject
corridor shot, converging lines to center
```

### 2.4 框架构图（Frame Within Frame）

**定义**：使用门框、窗户、栏杆等自然框架将主体框在其中。

**功能**：
- 囚禁感、被困感
- 偷窥感、客观观察
- 视觉隔离

**经典案例**：《教父》中门框构图、《肖申克的救赎》监狱栏杆

**AI实现**：
```
framed by doorway, frame within frame, claustrophobic
viewed through iron bars, trapped feeling
```

### 2.5 景深调度（Deep Space Composition）

**定义**：在Z轴的不同深度位置放置重要元素。

**功能**：
- 一个镜头内传达多重信息
- 角色之间的关系通过景深位置表达
- 观众可以选择看哪里（增加重看价值）

**经典案例**：《公民凯恩》中母亲在窗内签协议→小凯恩在窗外玩耍（三个角色在三个景深层次）

**AI实现**：
```
deep space composition, layered foreground/midground/background
character in foreground close, second character in deep background
```

### 2.6 负空间构图（Negative Space）

**定义**：主体周围留下大量空白区域。

**功能**：
- 孤独感（巨大的环境/微小的人物）
- 期待感（空白区域即将有东西进入）
- 注意力集中

**AI实现**：
```
negative space composition, vast empty space around subject, isolation
```

### 2.7 色彩对比构图（Color Contrast）

**定义**：利用互补色或对比色创造视觉冲击。

**经典配色**：
- 青橙调（Teal & Orange）：最受欢迎的好莱坞配色——冷背景vs暖肤色
- 红黑/红蓝：权力与冲突
- 黄紫：诡异的超现实感

**AI实现**：
```
teal and orange color palette, complementary colors
high contrast color composition, [color A] against [color B]
```

## 三、竖屏构图中的特殊改造

三分法在竖屏中需要调整为"垂直三分法"：

| 区域 | 内容 | 在文档中的应用 |
|---|---|---|
| 上1/3 | 环境/权力者/威胁源 | 铁窗、龙影、母姐站立位 |
| 中1/3 | 主要叙事焦点 | 对话、动作、表情 |
| 下1/3 | 受击位/残骸/细节 | 血迹、碎布、冰霜、跪着的人物 |

## 四、场景构图检查清单

```
□ 三分法还是对称构图？
    → 三分法 = 动态、不安、自然
    → 对称 = 权力、庄重、仪式感
□ 是否有引导线？
    → 没有引导线 → 观众视线可能浪费
□ 是否有框架构图？
    → 在竖屏囚禁类场景中特别有效
□ 前景/中景/后景是否都有内容？
    → Z轴三层信息（至少两层）
□ 颜色是否服务于情绪？
    → 不是"好看"而是"传达信息"
□ Y轴权力位置是否正确？
    → 上位=权力者 / 下位=受压迫者
```
