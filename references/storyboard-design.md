# Seedance 2.0 详细分镜图规则

## 1. 先分析剧情

先提取不可改写的剧情事实、原台词、人物关系、场景、道具、动作因果、情绪转折和时间预算，再拆成可拍摄的剧情节拍。一个格子只承载一个清晰的画面状态或主动作节点。

## 2. 选择分镜载体

- **标准故事板**：镜头较少，或需要为每镜保留较大的画面和详细信息栏；
- **九宫格**：约5–9个关键画面，适合单条15秒或结构清晰的短段落；不足9镜时保留空格或扩大重点格，不新增剧情凑格；
- **二十五宫格**：约16–25个关键画面，适合长段落、复杂动作或连续多条；不足25镜时不拆碎动作凑格；
- **智能宫格**：镜头数不适合固定九格/二十五格，或重点镜头需要跨格、放大、插入细节格时使用。按实际节拍动态选择格数、行列、格子大小和分页。

选择依据是剧情节拍、镜头数量、动作复杂度、对白时长和连续性风险，而不是视觉整齐。若一页信息过密，拆为连续多页并保持全局镜头编号连续。

无论选择哪种载体，每格信息标准完全相同。九宫格和二十五宫格不允许缩减字段；当单格无法容纳清晰画面、路线与信息栏时，必须提高画布分辨率、拆页或改用智能宫格。

## 3. 每格必须包含的信息

每个分镜格至少提供：

- 镜头序号 `S01、S02……` 与明确阅读顺序箭头；
- 所属视频条目和时间码/预计时长；
- 景别、机位高度、拍摄角度、焦段感和画幅；
- 主体构图、前中后景、角色ID、站位、朝向、视线和关键表情；
- 动作起点、运动路径、动作落点，以及摄影机运动箭头；
- 原台词归属、口型状态、环境音和关键动作音；
- 关键道具的位置、数量、持有者、左右手和状态；
- 主光方向、色温、时间、天气与重要反光；
- 与前后镜头的转场类型、继承状态和连续性锚点；
- 该格用于 Seedance 2.0 参考时必须保持的身份、动作、空间或尾帧信息。
- 视频使用状态：`USE`、`REFERENCE-ONLY` 或 `SKIP`，以及对应视频条目和时间范围。

## 4. 镜头使用状态与执行清单

- `USE`：最终视频中必须生成的镜头或画面状态；必须进入执行分镜图、视频时间线和最终提示词；
- `REFERENCE-ONLY`：不作为独立镜头生成，只用于角色身份、场景空间、动作衔接、起始帧或尾帧参考；在执行分镜图中使用明显的参考标签，并写明它服务哪个 `USE` 镜头；
- `SKIP`：备选、重复、被否决或不需要生成的镜头；从执行分镜图、最终提示词与 API 参考图中移除，只保留在独立弃用清单中说明原因；
- 状态不能用于删除原剧情必需动作、原台词或因果节点；若某剧情节点必须可见，它必须至少映射为一个 `USE` 镜头；
- 每次修改状态后重新检查镜头编号、时间码和连续性。执行镜头编号保持连续；弃用清单保留原规划ID并标 `SKIP`，不得占用执行时间。

输出镜头使用清单：

```text
S01｜USE｜第01条 0–3秒｜必须生成｜参考CHAR-001图片1、场景图片3
S02｜REFERENCE-ONLY｜服务S03起始状态｜不单独生成、不占时长
P03｜SKIP｜重复建立镜头｜不进入执行分镜图、不提交生成端
```

## 5. 图面设计

- 使用高分辨率、清晰分栏、统一画幅比例和足够大的单格；
- 把镜头编号置于每格左上角，把时间码和条目置于右上角；阅读箭头沿格间边界布置；
- 把文字标注放在格下信息栏或页边栏，不覆盖人脸、手部、道具和动作路径；
- 使用角色ID、颜色标签和一致图例，但不依赖颜色作为唯一识别方式；
- 动作箭头与摄影机箭头使用不同线型，并在图例中说明；
- 路线不得只靠颜色区分；同时使用代码、线型和箭头头型，确保灰度图或模型颜色识别不稳定时仍可判断；
- 分镜图允许制作标注；最终视频提示词仍默认无字幕、无画面文字。

## 6. AI 可辨识箭头与路线语法

每张分镜页固定放置以下图例，并在每条线上重复对应代码：

- `CAM` 摄影机运动：金色粗实线、双线箭头；标注 `CAM-START`、`CAM-END`、机位高度、速度和停止构图；
- `ACT` 角色或物体运动：青色实线、单线箭头；从动作起点连到落点，写角色/道具ID；
- `GAZE` 视线：白色细虚线、空心箭头；从眼睛指向注视对象；
- `FOCUS` 焦点/拉焦：蓝色点划线、菱形端点；写 `FOCUS A→B`；
- `LIGHT` 主光方向：橙色放射线、宽箭头；从光源指向受光面；
- `READ` 分镜阅读顺序：页边绿色编号箭头，只连接画格，不进入画格内容。

摄影机路线必须包含：

1. 起点与终点；
2. 推、拉、横移、升降、摇、俯仰、环绕、手持跟随或静止等动作类型；
3. 运动方向和空间路径；
4. 机位高度、朝向与轴线侧；
5. 匀速、加速、减速和停顿阶段；
6. 焦点是否随动；
7. 最终停止构图。

### 分段路线与后续方向

- 以 `CAM-A0` 表示起点，转向、升降、变速、停顿或重新构图处依次设置 `CAM-A1、CAM-A2……`，终点写 `CAM-A-END`；
- 每两个节点之间是一段独立路径，并在路径中段和接近下一节点处重复方向箭头；不能只在整条路线末端放一个箭头；
- 每段写时间范围与动作，例如 `0–2s PUSH IN / slow`、`2–4s ARC RIGHT / accelerate`、`4–5s CRANE UP / decelerate`；
- 转弯节点同时绘制进入箭头和离开箭头，离开箭头沿新方向切线放置，明确“到达这里后往哪里运动”；
- 升降使用垂直箭头和高度值，环绕使用角度/顺逆时针，推拉与横移写明相对角色或场景锚点；
- 复杂三维路线在画格旁增加俯视或侧视小图，与主画面共享完全相同的节点ID、时间段和箭头方向；
- 路径暂时停止但镜头未结束时使用 `HOLD` 节点，恢复运动必须从该节点画出新箭头；
- 镜头切换不能用连续路线跨越两个画格，必须在前镜 `END` 后由下一镜新的 `CAM-B0` 开始。

路线交叉时使用 `CAM-A、CAM-B` 与 `ACT-A、ACT-B` 分组，必要时拆成局部放大格。禁止使用没有起点、没有终点、没有逐段箭头、没有后续方向、没有归属或穿过主体导致歧义的装饰路线。

## 7. Seedance 2.0 参考约束

- 分镜图只描述确实要生成的镜头，不放装饰性画面或无关角色；
- 角色外观必须来自已经锁定的多视角角色设计，不得在不同格重新设计脸或服装；
- 保持同一场景的空间结构、轴线、光向和道具状态；
- 连续多条时标明每条的起始格、结束格、尾帧和下一条继承方式；
- 若平台只能接收有限数量参考图，优先提交角色板、当前条对应的分镜页和上一条尾帧，并在提示词中写明图片绑定关系；
- 输出分镜图后逐格对照剧情分析，确认没有漏镜、错序、改台词、人物混淆或连续性重置。
