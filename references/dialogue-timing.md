# 对白与时长设计

## 基础估算

普通中文对白可按约3.5–4.5字/秒估算，仅作为初步判断。

建议速度：

| 表演状态 | 建议速度 |
|---|---:|
| 清晰正常说话 | 3.8–4.5字/秒 |
| 平静低声 | 3.2–3.8字/秒 |
| 生气快速质问 | 4.0–5.0字/秒，但保留重音 |
| 哭腔、哽咽 | 2.0–3.2字/秒 |
| 老年、虚弱、喘息 | 2.0–3.0字/秒 |
| 机械播报 | 3.5–4.2字/秒 |

## 时间组成

一条视频的时间不只用于说话，还包括：

- 进入动作；
- 看向对方；
- 开口前呼吸；
- 句内停顿和重音；
- 对方反应；
- 镜头推进或切换；
- 结尾尾帧。

不要把总时长全部分配给台词。

## 台词规则

- 原文逐字保留；
- 角色归属明确；
- 电话音、门外音、画外音和内心独白标注声源；
- 内心独白人物不张嘴；
- 多人抢话时写清打断时刻和谁先停；
- 需要跨条时，在自然句末、动作落点或情绪转折处拆分；
- 禁止在半个词、半句话或语义未完成处切断。

## 15秒安全分配示例

- 0–2秒：空间与人物建立；
- 2–10.5秒：主要动作和对白；
- 10.5–13.5秒：转折或对方反应；
- 13.5–15秒：静默收束和尾帧。

实际分配以剧情为准。
