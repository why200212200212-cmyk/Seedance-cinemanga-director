# Seedance 2.0 详细分镜图模板

剧情分析完成后使用本模板。先选择分镜形式并锁定必要角色，再生成分镜图；不得为了填满固定宫格增加、删除或拆碎剧情。

## 分镜规划单

```text
【剧情节拍】
[节拍01 → 节拍02 → ……]

【分镜形式】
[标准故事板 / 九宫格 / 二十五宫格 / 智能宫格]

【选择依据】
- 必要镜头数：[N]
- 动作复杂度：[低 / 中 / 高]
- 对白与时间预算：[说明]
- 连续性风险：[角色 / 道具 / 空间 / 光影 / 尾帧]
- 排版方案：[行列、分页、重点格放大方式]

【阅读顺序】
S01 → S02 → …… → SNN

【镜头使用清单】
- S01｜USE｜[第几条、时间范围、必须生成原因、绑定参考图]
- S02｜REFERENCE-ONLY｜[服务哪个USE镜头、参考内容、不占生成时长]
- P03｜SKIP｜[弃用原因；不进入执行分镜图、不提交生成端]

【角色板绑定】
- CHAR-001｜角色名：[独立角色板/图片1；固定面部、发型、体型、服装；适用镜头]
- CHAR-002｜角色名：[独立角色板/图片2；固定面部、发型、体型、服装；适用镜头]

【场景板绑定】
- 场景01：[空间结构、轴线、主光、时间、天气]
```

## 每格镜头卡

为每个镜头逐项填写，不省略镜头序号和时间映射：

```text
【S01｜第01条｜00:00–00:03】
- 视频使用状态：[USE / REFERENCE-ONLY；SKIP镜头不得放入执行分镜图]
- 服务/映射：[对应视频条目与时间段；REFERENCE-ONLY服务哪个USE镜头]
- 画面：
- 景别/机位/角度/焦段感/画幅：
- 构图与前中后景：
- 角色ID/角色名/独立参考图编号/站位/朝向/视线/表情：
- 动作起点 → 路径 → 落点：
- 摄影机运动与焦点：
- 路线代码：[CAM-A / ACT-A / GAZE-A / FOCUS-A / LIGHT-A]
- 路线节点与分段：[CAM-A0 → CAM-A1 → CAM-A2 → CAM-A-END]
- 每段时间/动作/方向/速度/机位高度：[逐段填写]
- 转向后的下一方向：[在每个节点填写进入方向 → 离开方向]
- 俯视/侧视路线小图：[需要 / 不需要；节点ID与主画面一致]
- 原台词/口型/声音：
- 道具/左右手/状态：
- 光影/时间/天气：
- 转场类型：
- 从上一镜继承：
- 传给下一镜：
- Seedance 2.0 参考重点：
```

## 分镜图生成提示词

```text
生成一张高分辨率影视制作执行分镜图，采用[标准故事板/九宫格/二十五宫格/智能宫格]，共[N]个必要镜头。[横向/纵向]阅读，镜头编号从S01连续到SNN，格间使用明确阅读箭头；多页时页码和镜头号连续。所有分镜形式使用同一套完整信息字段；如果单格信息过密则扩大画布或拆页，不得省略字段。

每格左上角清晰显示镜头序号与USE/REFERENCE-ONLY状态，右上角显示所属视频条目与时间码；格下信息栏写景别、机位、焦段感、角色ID、动作、运镜、台词/声音、道具、光影、转场和连续性锚点。SKIP镜头不得出现在本执行分镜图中，只放入单独弃用清单。每页固定提供AI可辨识路线图例：CAM摄影机运动=金色粗实线双箭头，ACT角色/物体运动=青色实线单箭头，GAZE视线=白色细虚线空心箭头，FOCUS焦点变化=蓝色点划线菱形端点，LIGHT主光=橙色放射宽箭头，READ阅读顺序=页边绿色编号箭头。不得只依赖颜色，每条线重复写代码。

所有CAM路线使用连续编号节点，例如CAM-A0、CAM-A1、CAM-A2、CAM-A-END；转向、升降、变速和停顿处必须设节点。每个节点间重复方向箭头并标时间、动作类型、速度和机位高度；转弯处同时画进入箭头与沿新方向离开的箭头，明确到达节点后的继续方向。不能只在整条线末端放一个箭头。复杂三维路线增加俯视或侧视小图，与主画面共享相同节点ID。路线暂停使用HOLD节点，恢复时从HOLD节点绘制新箭头。

复杂或交叉路线使用CAM-A、CAM-B分组，必要时增加局部放大格。所有标注位于边框或信息栏，不遮挡人脸、手部、关键道具和动作路径。

角色严格继承已绑定的独立多视角角色板；每个人像附近显示永久角色ID，格下信息栏重复角色名和独立参考图编号。同一角色跨格不变脸，不同角色不混脸。场景结构、轴线、主光方向、道具数量和持有关系保持连续。只呈现剧情分析确认的镜头，不添加装饰性人物、道具或剧情。

[逐格粘贴“每格镜头卡”的画面与执行信息。]
```

## 输出前检查

- 镜头号、时间码、条目映射和阅读顺序无歧义；
- USE、REFERENCE-ONLY和SKIP清单明确；执行图不含SKIP，REFERENCE-ONLY写明服务对象且不占生成时长；
- 四种分镜形式均保留完整字段，未因格数增加而缩减信息；
- CAM、ACT、GAZE、FOCUS、LIGHT、READ图例完整，任何路线在不看颜色时仍可识别；
- 摄影机路线具有明确起点、终点、归属、方向、速度阶段、机位高度和停止构图；
- 每次转向、升降、变速或停顿均有连续节点，每一段都有箭头和下一方向，俯视/侧视小图与主画面节点一致；
- 所有原台词逐字一致；
- 分镜格数等于必要画面节点数，不为填格改变剧情；
- 每格均可映射到 Seedance 2.0 提示词中的一个镜头或关键状态；
- 角色板、场景板、上一条尾帧和当前分镜页的图片编号绑定清楚；
- 每个角色拥有单独角色板和永久ID，多人总览图没有被当作身份参考图；
- 分镜标注只服务制作参考，不成为最终视频字幕或画面文字。
