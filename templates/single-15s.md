# 完整15秒单条输出模板

仅在用户要求“完整15秒、单条、不拆分、只输出一条”时使用。

## 输出前内部校验

- 原剧情与原台词未改变；
- 出场人物仅包含当前段落必要角色；
- 台词能在15秒内自然说完；
- 结尾至少保留0.5–1.5秒反应与尾帧；
- 人物、服装、道具、空间、光影已锁定；
- 无音乐、无BGM、无字幕、无水印，除非用户另有要求。

若台词不能自然完成且用户明确禁止拆分，不生成缩水版本；先返回固定冲突报告：“最低自然时长约X秒；请选择缩短台词、延长时长或允许拆分。”

## 固定成品格式

```text
_::~OUTPUT_START::~_

【模式】
完整15秒，单条视频；[精致3D国漫影视级 / 真人影视级]。

【导演方案（审阅用，不粘贴到生成器）】

【可复制图片提示词】
[只描述能够代表本条视频的静态关键画面。写清人物外观、服装、站位、场景结构、关键道具、光源方向、色温、构图和风格。不要写连续动作。]

===

【可复制视频提示词】
即梦Seedance视频，完整15秒，单条视频，有音效，无音乐，无BGM，无字幕，无水印。除时间轴明确描述的变化外，其余角色身份、服装、空间、道具、光影和天气状态保持连续；运动质感、摄影机稳定度与动作节奏服从本镜剧情设计。

【人物与连续性锁定】
- 必须出现：[角色]
- 允许背景：[无 / 角色]
- 禁止出现：[角色或“其他无关人物与动物”]
- 固定服装：[描述]
- 固定空间：[门窗、家具、前中后景、人物初始站位]
- 固定道具：[数量、位置、持有者、左右手、状态]
- 固定光影：[主光方向、色温、明暗、眼神高光、道具反光]

【摄影与物理合同】
- 画幅与摄影观感：[画幅；真人模式写等效焦段/运镜设备，3D模式写虚拟摄影机观感]

- 模式专项：[3D仅写身份锚点、材质响应、绑定/关节、形变接触与次级运动；真人仅写帧率/快门观感、等效焦段、设备、曝光/白平衡、焦点路径、实际灯源、演员目标与阻碍]

【0–t1秒｜节拍1】
- 景别与机位：[景别、机位高度、角度]
- 镜头运动：[起点、路径、速度、停止点]
- 人物站位与面对关系：[左/右/中，前/中/后景，面对谁]
- 动作顺序：[动作1 → 动作2 → 动作3]
- 台词：[角色：“原文台词” / 无]
- 配音：[语气、音量、速度、是否张嘴]
- 音效：[环境与动作音]
- 镜头结束状态：[构图与姿态]

【t1–t2秒｜节拍2】
- 景别与机位：[...]
- 镜头运动：[...]
- 人物站位与面对关系：[...]
- 动作顺序：[...]
- 台词：[...]
- 配音：[...]
- 音效：[...]
- 镜头结束状态：[...]

【t2–15秒｜节拍3】
- 景别与机位：[...]
- 镜头运动：[...]
- 人物站位与面对关系：[...]
- 动作顺序：[...]
- 台词：必须在尾帧前完成
- 配音：[...]
- 音效：[...]

【负面约束】
[只列本条实际高风险的3–8项，例如：禁止改动原台词、角色身份漂移、手与关键道具无接触、脚底滑行、焦点无因抽动。不要复制整份通用禁令。]

_::~OUTPUT_END::~_
```

## 一镜到底变体

可以使用一镜到底，但必须在同一条视频提示词中依次说明：

- 0–t1秒：建立空间与人物关系；
- t1–t2秒：动作或对白推进；
- t2–15秒：转折、反应和尾帧；
- 镜头路径必须连续、可实现、不穿透物体。
