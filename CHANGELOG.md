# Changelog

## 2026-07-15 — Quality hardening

- 新增零依赖火山方舟视频任务客户端，支持创建、查询、等待、取消、尾帧请求和安全下载；模型与推理接入点保持可配置。
- 新增 Codex、OpenClaw 与其他 AgentSkills 宿主的统一编排契约、API 配置文档、dotenv 示例和 dry-run 示例。
- 新增离线客户端单元测试与 CI 请求预检；CI 不持有密钥、不联网创建任务，也不会产生生成费用。
- 清理未配置且与本项目技术栈无关的商业安全扫描模板；保留 CodeQL、Bandit 与项目原生验证，并把检出动作更新到当前主版本。
- 新增 GitHub Actions，在 Python 3.10 与 3.13 上自动校验技能、文档链接和视觉资产；
- 将风格参考收缩为模式决策路由，避免3D与真人专项规则重复加载；
- 补充不可满足的单条时长冲突、合法换场、匹配剪辑、复杂度过载、模式隔离和平台能力未知等验收用例；
- 把 CI 工作流和关键边界用例纳入结构校验。
